# 从视频到字幕：使用Whisper实现日语语音识别与翻译

今天完成了一个有趣的项目：使用OpenAI的Whisper模型，将日文视频转换为中文字幕。整个过程涉及音频提取、语音识别、翻译等多个环节，遇到了不少技术挑战，最终都一一解决。下面分享一下完整的实现过程和踩坑经验。

## 项目背景

需要将一部日文动画视频（MKV格式）提取音频，识别生成日语字幕，然后翻译成中文字幕。视频文件路径：`{user_downloads}/Frieren_S02E07.mkv`

## 技术方案

### 1. 音频提取

使用ffmpeg从视频中提取音频：

```bash
ffmpeg -i input.mkv -vn -acodec copy audio.opus
```

参数说明：
- `-vn`：不包含视频流
- `-acodec copy`：直接复制音频编码，不重新编码

### 2. 语音识别

使用Whisper模型进行日语语音识别。Whisper是OpenAI开源的多语言语音识别模型，支持99种语言。

**硬件配置检查**：
- CPU：Intel Core i7
- 内存：16GB
- GPU：NVIDIA显卡（16GB显存）
- 操作系统：Windows

**模型选择**：
根据硬件配置，推荐使用`large`模型（约2.88GB），在保证识别质量的同时，16GB显存可以轻松加载。

### 3. 字幕翻译

将识别的日语字幕翻译为中文，并纠正识别错误。

import torch
import os

def transcribe_japanese(audio_file, model_size="large"):

    print(f"正在加载 {model_size} 模型到 {device}...")

        language="Japanese",

    )
    
    return result

## 遇到的技术挑战

### 1. 模型下载校验失败

**错误信息**：
```
RuntimeError: Model has been downloaded but the SHA256 checksum does not match.
```

**解决方案**：
删除损坏的模型文件，重新下载：
```bash
Remove-Item "~/.cache/whisper/large-v3.pt" -Force
```

### 2. 代码语法错误

**错误信息**：
```
AttributeError: module 'whisper.utils' has no attribute 'write_srt'
```

**原因**：函数名大小写错误

**解决方案**：
```python
# 错误
f.write(whisper.utils.write_srt(result["segments"]))

# 正确
f.write(whisper.utils.WriteSRT(result["segments"]))
```

或者自己实现SRT生成函数：
```python
def write_srt(segments):
    srt_content = []
    for i, segment in enumerate(segments, start=1):
        start_time = format_timestamp(segment['start'])
        end_time = format_timestamp(segment['end'])
        text = segment['text'].strip()
        
        srt_content.append(f"{i}")
        srt_content.append(f"{start_time} --> {end_time}")
        srt_content.append(f"{text}")
        srt_content.append("")
    
    return "\n".join(srt_content)
```

### 4. 翻译质量不佳

**问题**：翻译结果存在未翻译成功和语句不通顺的问题

**初步解决方案**：
- 扩充翻译词典，增加常见词汇、短语、动词、形容词
- 添加日语助词和结尾词的替换规则
- 优化翻译逻辑，提高覆盖率

**最终解决方案**：使用专业翻译API

基于词典的翻译方法存在局限性，无法处理复杂的语法结构和上下文关系。因此，决定使用专业翻译API来提高翻译质量。

## 使用翻译API优化字幕翻译

### 翻译API选择

对比了多个翻译服务后，选择了两种方案：

| API | 翻译质量 | 价格 | 免费额度 | 推荐指数 |
|-----|---------|------|---------|---------|
| DeepL | ⭐⭐⭐⭐⭐ | 中等 | 50万字符/月 | ⭐⭐⭐⭐⭐ |
| MyMemory | ⭐⭐⭐ | 免费 | 500次/小时 | ⭐⭐⭐ |

**最终选择**：DeepL API（翻译质量最高，尤其擅长日语-中文翻译）

### 实现翻译脚本

创建`translate_with_api.py`脚本，实现以下功能：

```python
import requests
import re
import time

class DeepLTranslator:
    """DeepL翻译器"""
    
    def __init__(self, auth_key):
        self.auth_key = auth_key
        self.base_url = "https://api-free.deepl.com/v2/translate"
        self.headers = {"Authorization": f"DeepL-Auth-Key {auth_key}"}
    
    def translate(self, text, source_lang="JA", target_lang="ZH"):
        """翻译文本"""
        if not text.strip():
            return text
        
        data = {
            "text": [text],
            "source_lang": source_lang,
            "target_lang": target_lang
        }
        
        response = requests.post(
            self.base_url,
            headers=self.headers,
            json=data
        )
        
        result = response.json()
        return result["translations"][0]["text"]


def parse_srt_file(input_file):
    """解析SRT文件"""
    with open(input_file, 'r', encoding='utf-8') as f:
        content = f.read()
    
    blocks = re.split(r'\n\s*\n', content.strip())
    subtitles = []
    
    for block in blocks:
        lines = block.strip().split('\n')
        if len(lines) >= 3:
            subtitles.append({
                'index': lines[0].strip(),
                'timestamp': lines[1].strip(),
                'text': '\n'.join(lines[2:])
            })
    
    return subtitles


def translate_srt_file(input_file, output_file, translator):
    """翻译SRT文件，翻译一句写入一句"""
    subtitles = parse_srt_file(input_file)
    
    with open(output_file, 'w', encoding='utf-8') as f:
        for i, subtitle in enumerate(subtitles, 1):
            # 翻译文本
            translated_text = translator.translate(subtitle['text'])
            
            # 立即写入翻译结果
            f.write(subtitle['index'] + '\n')
            f.write(subtitle['timestamp'] + '\n')
            f.write(translated_text + '\n')
            f.write('\n')
            
            # 确保写入磁盘
            f.flush()
            
            # 避免API调用过快
            if i % 10 == 0:
                time.sleep(1)
```

### 关键优化点

1. **实时写入**：翻译一句就立即写入文件，即使程序中断也不会丢失已翻译的内容
2. **flush缓冲**：使用`f.flush()`确保数据写入磁盘
3. **内存优化**：不需要在内存中保存所有翻译结果
4. **错误处理**：API调用失败时保留原文，避免整个翻译失败

### 使用方法

```bash
python translate_with_api.py
```

选择翻译服务并输入API密钥（如使用DeepL），脚本会自动完成翻译。

## 经验总结

1. **模型选择很重要**：根据硬件配置选择合适的模型，large模型在16GB显存下表现最佳
2. **参数调优很关键**：合理设置Whisper的识别参数可以显著提高质量
