🚀 脚本管理器 API 文档

完整的 API 端点参考指南

📜 脚本管理 API

GET /api/scripts ▼

获取脚本列表

返回所有脚本的详细信息，包括名称、大小、描述等。

响应格式

[
{ "name": "example.py", "path": "/app/scripts/example.py", "size": 1024, "mtime": 1617296400.0, "description": "示例脚本" } ]
POST /api/execute ▼

执行脚本

执行指定的脚本，并返回任务 ID。

请求参数

参数 类型 必填 描述
script string 是 脚本名称
args string 否 脚本参数
email_on_success boolean 否 执行成功时发送邮件
email_on_failure boolean 否 执行失败时发送邮件

示例请求

{ "script": "example.py", "args": "--help", "email_on_success": true, "email_on_failure": true }

响应格式

{ "message": "脚本已加入执行队列", "task_id": "abc123" }
POST /api/script/upload

上传脚本

上传 Python 脚本文件到服务器。

请求参数

参数 类型 必填 描述
file file 是 Python 脚本文件 (.py)

响应格式

{ "message": "脚本上传成功", "script": { "name": "example.py", "path": "/app/scripts/example.py", "size": 1024, "mtime": 1617296400.0, "description": "" } }
POST /api/script/delete/{script_name}

删除脚本

删除指定的脚本文件。

响应格式

{ "message": "脚本删除成功" }
GET /api/script/description/{script_name}

获取脚本描述

获取指定脚本的描述信息。

响应格式

{ "description": "示例脚本" }
POST /api/script/description/{script_name}

更新脚本描述

更新指定脚本的描述信息。

请求参数

参数 类型 必填 描述
description string 是 脚本描述

示例请求

{ "description": "这是一个示例脚本" }

响应格式

{ "message": "脚本简介更新成功" }
GET /api/script/view/{script_name}

查看脚本内容

查看指定脚本的完整内容。

响应格式

{ "content": "print('Hello, World!')" }
POST /api/scripts/batch-delete

批量删除脚本

批量删除多个脚本文件。

请求参数

参数 类型 必填 描述
scripts array 是 脚本名称数组

示例请求

{ "scripts": ["script1.py", "script2.py"] }

响应格式

{ "message": "成功删除 2 个脚本" }

⚙️ 执行队列 API

GET /api/queue/tasks

获取任务列表

返回所有执行任务的详细信息。

响应格式

[
{ "id": "abc123", "script_name": "example.py", "script_path": "/app/scripts/example.py", "args": "--help", "status": "已完成", "log_file": "/app/logs/example.py_20230401_120000_abc123.log", "start_time": "2023-04-01T12:00:00", "end_time": "2023-04-01T12:00:05", "return_code": 0 } ]
GET /api/queue/status

获取队列状态

返回执行队列的状态信息，包括等待、运行和完成的任务数量。

响应格式

{ "pending": 0, "running": 1, "completed": 5, "total": 6, "max_concurrent": 3 }
GET /api/queue/task/{task_id}

获取任务详情

返回指定任务的详细信息。

响应格式

{ "id": "abc123", "script_name": "example.py", "script_path": "/app/scripts/example.py", "args": "--help", "status": "已完成", "log_file": "/app/logs/example.py_20230401_120000_abc123.log", "start_time": "2023-04-01T12:00:00", "end_time": "2023-04-01T12:00:05", "return_code": 0 }
POST /api/queue/stop/{task_id}

停止任务

停止正在运行的任务。

响应格式

{ "success": true, "message": "任务已停止" }
POST /api/queue/restart/{task_id}

重新执行任务

重新执行指定的任务。

响应格式

{ "success": true, "task_id": "def456" }
GET /api/queue/running

获取运行中的任务

返回当前正在运行的任务列表。

响应格式

[
{ "id": "abc123", "script_name": "example.py", "script_path": "/app/scripts/example.py", "args": "--help", "status": "运行中", "log_file": "/app/logs/example.py_20230401_120000_abc123.log", "start_time": "2023-04-01T12:00:00", "end_time": null, "return_code": null } ]

⏰ 定时任务 API

GET /api/jobs

获取任务列表

返回所有定时任务的详细信息。

响应格式

{ "job1": { "id": "job1", "script": "example.py", "cron": "0 0 * * *", "enabled": true, "args": "", "email_on_success": false, "email_on_failure": true } }
POST /api/job/add

添加定时任务

添加新的定时任务。

请求参数

参数 类型 必填 描述
id string 是 任务 ID
script string 是 脚本名称
cron string 是 Cron 表达式
enabled boolean 否 是否启用
args string 否 脚本参数
email_on_success boolean 否 执行成功时发送邮件
email_on_failure boolean 否 执行失败时发送邮件

示例请求

{ "id": "job1", "script": "example.py", "cron": "0 0 * * *", "enabled": true, "args": "--help", "email_on_success": false, "email_on_failure": true }

响应格式

{ "message": "任务添加成功" }
POST /api/job/delete

删除定时任务

删除指定的定时任务。

请求参数

参数 类型 必填 描述
id string 是 任务 ID

示例请求

{ "id": "job1" }

响应格式

{ "message": "任务删除成功" }
POST /api/job/update

更新定时任务

更新指定的定时任务。

请求参数

参数 类型 必填 描述
id string 是 任务 ID
script string 是 脚本名称
cron string 是 Cron 表达式
enabled boolean 否 是否启用
args string 否 脚本参数
email_on_success boolean 否 执行成功时发送邮件
email_on_failure boolean 否 执行失败时发送邮件

示例请求

{ "id": "job1", "script": "example.py", "cron": "0 1 * * *", "enabled": true, "args": "--help", "email_on_success": true, "email_on_failure": true }

响应格式

{ "message": "任务更新成功" }
POST /api/jobs/batch-delete

批量删除定时任务

批量删除多个定时任务。

请求参数

参数 类型 必填 描述
jobs array 是 任务 ID 数组

示例请求

{ "jobs": ["job1", "job2"] }

响应格式

{ "message": "成功删除 2 个任务" }
POST /api/jobs/batch-enable

批量启用/禁用定时任务

批量启用或禁用多个定时任务。

请求参数

参数 类型 必填 描述
jobs array 是 任务 ID 数组
enabled boolean 是 是否启用

示例请求

{ "jobs": ["job1", "job2"], "enabled": true }

响应格式

{ "message": "成功更新 2 个任务" }

📊 执行日志 API

POST /api/logs/delete/{filename}

删除日志

响应格式

{ "message": "日志删除成功" }
POST /api/logs/delete

清空所有日志

删除所有日志文件。

响应格式

{ "message": "日志清空成功" }

✉️ 邮件设置 API

GET /api/email/settings

获取邮件设置

返回当前的邮件设置。

响应格式

{ "smtp": "smtp.gmail.com", "port": 587, "from": "your-email@gmail.com", "password": "your-password", "to": "recipient@example.com", "ssl": true, "subject": "脚本执行通知 - {status}", "template": "脚本名称: {script_name}\n执行状态: {status}\n执行时间: {time}\n\n日志内容:\n{log}" }
POST /api/email/settings

保存邮件设置

保存邮件设置。

请求参数

参数 类型 必填 描述
smtp string 是 SMTP 服务器
port number 是 SMTP 端口
from string 是 发件人邮箱
password string 是 邮箱密码/授权码
to string 是 收件人邮箱
ssl boolean 否 使用 SSL/TLS
subject string 否 邮件主题模板
template string 否 邮件内容模板

示例请求

{ "smtp": "smtp.gmail.com", "port": 587, "from": "your-email@gmail.com", "password": "your-password", "to": "recipient@example.com", "ssl": true, "subject": "脚本执行通知 - {status}", "template": "脚本名称: {script_name}\n执行状态: {status}\n执行时间: {time}\n\n日志内容:\n{log}" }

响应格式

{ "message": "邮件设置保存成功" }
POST /api/email/test

测试邮件发送

发送测试邮件，验证邮件设置是否正确。

响应格式

{ "message": "测试邮件发送成功" }
